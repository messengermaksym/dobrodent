# Стоматологічна клініка "Добродент" — Вебсайт та CMS

Сучасний, швидкий та адаптивний вебсайт для стоматологічної клініки **"Добродент"**. Проєкт розроблено на основі новітнього веб-стеку з фокусом на високу швидкість завантаження, SEO-оптимізацію, преміальний дизайн та наявність зручної адмін-панелі для керування контентом.

---

## Використані технології

### **Core & Framework**
- **[Next.js 16 (App Router)](https://nextjs.org/)** — фреймворк для статичної генерації (SSG) та оптимізації продуктивності.
- **[React 19](https://react.dev/)** — бібліотека для побудови інтерфейсу користувача.
- **[TypeScript](https://www.typescriptlang.org/)** — сувора типовізація для надійності коду.

### **Стилізація та Анімація**
- **[Tailwind CSS v4](https://tailwindcss.com/)** — утилітарний CSS-фреймворк для швидкої та адаптивної верстки.
- **[Framer Motion](https://www.framer.com/motion/)** — плавні мікро-анімації появи елементів при скролі (`ScrollReveal`, `StaggerContainer`).
- **[Lucide React](https://lucide.dev/)** — набір сучасних векторних іконок.
- **Google Fonts** — типографіка: `Cormorant Garamond` (заголовки), `Montserrat` (акценти/меню), `Inter` (основний текст).

### **Інтеграції та CMS**
- **GitHub API / Headless CMS** — збереження змін із адмін-панелі безпосередньо у репозиторій.
- **[Firebase](https://firebase.google.com/)** — інтеграція баз даних та баклієнту.
- **[Cloudflare Turnstile](https://www.cloudflare.com/products/turnstile/)** — сучасний захист від ботів та спаму (без дратуючих капч).

## Реалізований функціонал

- **Головна сторінка (`/`):**
  - Естетичний Hero-блок з закликом до дії та кнопкою швидкого запису.
  - Секція переваг клініки, популярних послуг та лікаря.
  - Блок живих відгуків пацієнтів з оцінками 5⭐.
  - Секція поширених запитань (FAQ) з розгортальними відповідями.
- **Прайс-лист (`/prices`):** Прозорий та структурований каталог послуг за категоріями з можливістю швидкого запису.
- **Команда фахівців (`/specialists`):** Інформаційні картки лікарів (досвід, спеціалізація, фотографія).
- **Галерея робіт (`/gallery`):** Фотогалерея результатів лікування, кабінетів та обладнання.
- **Контакти (`/contact`):** Форма зворотного зв'язку, карта розташування, години роботи та контактні номери.
- **Модальне вікно запису (`BookingModal`):** Швидкий запис на прийом з будь-якої сторінки без перезавантаження.
- **Адмін-панель (`/admin`):** 
  - Захищена зона для автономного редагування цінового прайсу, даних лікарів та галереї.
  - Вбудований захист від брутфорсу (тимчасова блокування при 3 невдалих спробах).
- **SEO & Продуктивність:**
  - Повна адаптивність (Mobile-First дизайн).
  - Налаштовані OpenGraph мета-теги, `sitemap.xml` та `robots.txt`.
  - Максимальні показники швидкості в Google PageSpeed Insights / Lighthouse.

## Як запустити проєкт локально

### **Передумови**
Переконайтеся, що у вас встановлено [Node.js](https://nodejs.org/) (версія 18.x або новіша) та `npm`.

### **Кроки встановлення**

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/your-username/dobrodent.git
   cd dobrodent
   ```

2. **Перейдіть у папку веб-додатка:**
   ```bash
   cd dobrodent-web
   ```

3. **Встановіть залежності:**
   ```bash
   npm install
   ```

4. **Запустіть сервер розробки:**
   ```bash
   npm run dev
   ```

5. **Відкрийте браузер:**
   Перейдіть за адресою [http://localhost:3000](http://localhost:3000).
