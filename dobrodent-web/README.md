# Стоматологічна клініка "Добродент" — Web Application

Ця папка містить вихідний код вебсайту та CMS стоматологічної клініки **"Добродент"**, побудований на **Next.js 16 (App Router)** та **React 19**.

Детальну документацію по проєкту дивіться у головному файлі [README.md](../README.md).

---

## Швидкий запуск

```bash
# Встановлення залежностей
npm install

# Запуск локального сервера розробки
npm run dev

# Збірка продакшен версії
npm run build
```

Додаток буде доступний за адресою [http://localhost:3000](http://localhost:3000).
